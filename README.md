# Spring Boot Maven News application

##  What you'll need

     JDK 1.7 or later
     Maven 3 or later
     spring-boot 1.3.5.RELEASE

##  Build
     
    mvn clean install    

##  Packaging

     mvn package 

##  Test

     mvn test   


##  Run

     mvn spring-boot:run


##  Used technologies
	
	AngularJS
	Springboot
	JWT kotens
	Maven
	Docker
